# Le Dallas — Steakhouse & Grill

Site vitrine d'un restaurant steakhouse (Argenteuil), avec authentification et réservations via Supabase.

## Contenu

- `index.html` — site vitrine une page (hero, carte, infos, réservation)
- `auth.html` — démo d'authentification Supabase (email + mot de passe)

## Stack

- HTML / CSS / JavaScript vanilla, zéro build
- [Supabase](https://supabase.com) pour l'auth et le stockage des réservations

## Config

Renseigner dans le code le `SUPABASE_URL` et la clé publique (`anon` / `publishable`)
du projet Supabase. La clé secrète ne doit jamais être commit.
